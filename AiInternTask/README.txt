# Image Segmentation using TensorFlow and OpenCV

This repository contains a project for performing image segmentation using a pre-trained TensorFlow model and OpenCV.

# Project Structure

- `saved_model/`: Directory containing the pre-trained TensorFlow model.
- `image.jpg`: Sample image used for segmentation.
- `image_segmentation.py`: Python script for performing image segmentation.

# Requirements

- Python 3.x
- TensorFlow
- OpenCV
- NumPy

# Setup

- Clone the repository to your local machine.
- Run image_segmentation.py
